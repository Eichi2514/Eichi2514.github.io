경비 양식 사용법 (5단계)

1

모임 생성

메인페이지 상단에 "시작" 버튼을 눌러 양식 선택 후 모임 이름을 입력하면 해당 페이지로 이동합니다.
2

기본 정보 입력

참여자 "추가" 버튼을 눌러 이름과 선납금액을 입력하세요.
3

지출 내역 입력

지출 "추가" 버튼을 눌러 지출 항목과 금액을 입력하세요.
4

결과 확인

개인별 정산금액(분담금)을 확인하세요.
5

이전 모임 기록 확인

사이트를 나갔다가 다시 방문하면 이전에 작성한 모임 기록이 목록에 저장됩니다. 모임 목록에서 클릭하여 다시 확인할 수 있습니다.

회비 양식 사용법 (7단계)

1

모임 생성

메인페이지 상단에 "시작" 버튼을 눌러 양식 선택 후 모임 이름을 입력하면 해당 페이지로 이동합니다.
2

기본 정보 입력

참여자 "추가" 버튼을 눌러 멤버의 이름을 입력하세요.
3

인원 관리

멤버별로 "납부" 및 "참여" 버튼을 사용해 회비 납부 여부와 모임 참여 여부를 관리할 수 있습니다. 납부한 날짜는 수정할 수 있지만, '연도-월(YYYY-MM)' 형식으로 입력해야 정상적으로 적용됩니다.
4

지출 내역 입력

지출 "추가" 버튼을 눌러 지출 항목과 금액을 입력하세요.
5

결과 확인

월별 지출, 남은 잔액, 참여 인원이 자동 계산되어 표시됩니다.
6

날짜 변경

좌측 상단의 달력을 클릭하면 원하는 날짜로 이동할 수 있습니다. 새로운 월에 데이터가 없을 경우, 이전 달의 기본 정보를 자동으로 불러옵니다.
7

이전 모임 기록 확인

사이트를 나갔다가 다시 방문하면 이전에 작성한 모임 기록이 목록에 저장됩니다. 모임 목록에서 클릭하여 다시 확인할 수 있습니다.

자주 묻는 질문 (FAQ)

어떻게 사용하나요?

화면 상단의 "시작" 버튼을 누른 후, 양식 선택을 선택하고 모임의 제목을 입력하면 됩니다.

경비양식) 정산금액(분담금) 계산 방식은 어떻게 되나요?

기본적으로 1/N로 균등하게 분할합니다.
이후, 계산된 분할 금액에서 각자의 선납 금액을 반영한 후 최종 금액이 표시됩니다.

경비양식) 정산금액이 음수(-)로 표시되는 이유는 무엇인가요?

정산금액이 음수(-)라면 돌려받을 금액이며, 양수(+)라면 내야 할 금액을 의미합니다.

회비양식) 멤버가 납부한 금액을 개별적으로 입력할 수 있나요?

현재 모든 멤버가 동일한 회비를 납부하는 방식으로 설계되어 있습니다.

회비양식) 납부 날짜를 입력했는데 적용이 안돼요

입금일을 '연도-월(YYYY-MM)' 형식으로 입력해야 정상적으로 적용됩니다.
