모임 목록

주요 기능

1

사용하기 쉬운 인터페이스

직관적인 디자인으로 누구나 쉽게 사용 할 수 있습니다.

2

내역 관리

간편하게 지출 내역을 입력하고 관리할 수 있습니다.

3

자동 계산

정확하고 빠르게 정산금액(분담금)을 계산합니다.

사용 방법 (5단계)

1

모임 생성

메인페이지 상단에 "시작" 버튼을 눌러 모임 이름을 입력하면 모임 페이지로 이동합니다.
2

기본 정보 입력

참여자 "추가" 버튼을 눌러 이름과 선납금액을 입력하세요.
3

지출 내역 입력

지출 "추가" 버튼을 눌러 지출 항목과 금액을 입력하세요.
4

결과 확인

개인별 정산금액(분담금)을 확인하세요.
5

이전 모임 기록 확인

사이트를 나갔다가 다시 방문하면 이전에 작성한 모임 기록이 목록에 저장됩니다. 모임 목록에서 클릭하여 다시 확인할 수 있습니다.

자주 묻는 질문 (FAQ)

어떻게 사용하나요?

화면 상단의 "시작" 버튼을 누른 후, 모임의 제목을 입력하면 됩니다.

정산금액(분담금) 계산 방식은 어떻게 되나요?

기본적으로 1/N로 균등하게 분할합니다.
이후, 계산된 분할 금액에서 각자의 선납 금액을 반영한 후 최종 금액이 표시됩니다.

정산금액이 음수(-)로 표시되는 이유는 무엇인가요?

정산금액이 음수(-)라면 돌려받을 금액이며, 양수(+)라면 내야 할 금액을 의미합니다.
