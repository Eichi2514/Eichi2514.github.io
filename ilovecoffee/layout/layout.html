  2. 머신 수 설정 : 오른쪽 패널에서 로머, 쿠머, 쇼케 등의 수를 조절하세요.

  4. 되돌리기 : 배치 시작 전 상태로 돌아갑니다.
  5. 초기화 : 모든 기타칸과 머신 수를 초기화하고 새로 시작합니다.
